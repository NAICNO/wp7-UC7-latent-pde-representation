# Results

This directory stores training outputs generated by the UC7 pipeline:

- **Training logs** — loss curves, learning rate schedules
- **Evaluation metrics** — reconstruction errors (MSE, relative L2)
- **Plots** — latent space visualizations, reconstructed fields, comparison figures
- **Exported data** — latent embeddings, alignment mappings

Files in this directory are generated by the scripts in `src/` and by the
`demonstrator_pde.ipynb` notebook. They are not committed to version control
(see `.gitignore`).

Quick reference

.. important::

Setup Script
------------

Run after cloning to set up the environment:

.. code-block:: bash

   git clone https://github.com/NAICNO/wp7-UC7-latent-pde-representation.git
   cd wp7-UC7-latent-pde-representation
   ./setup.sh
